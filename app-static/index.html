Star Digger

Explore the Maze to Find What Lies Beneath the Ancient Structures!

Sri's submission for NEW JAMSHIRE 2024

click screen to focus input, then use W A S D keys to move around.

Star Digger is a submission for the 2024 New JAMshire Underworld Game Jam.

This is hacked-up movement demo to help get a feeling for the space. It's based on elements of Star Maze from 1982, one of my favorite childhood game memories.

The framework is a combination of ThreeJS and TypeScript, loosely inspired by XNA's component-style module architecture.

Submitted November 20, 2022

KEYS: W A S D (click screen to focus input)

Source code at github/dsriseah/jamshire-underworld. A running version is located at dsriseah.com/_staging/star-digger.
