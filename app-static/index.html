About

Underworld is a submission for the 2024 New JAMshire Game Jam, inspired by elements of Star Maze from 1982. Framework is based on ThreeJS and written in Typescript to resemble XNA's component-based modular design.

Source github/dsriseah/jamshire-underworld. A running version is located at dsriseah.com/_staging/star-digger.
